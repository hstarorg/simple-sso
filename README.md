# simple-sso
A simple  single sign on project.

# Usage

```bash
# Clone the project
git clone

# Install dependencies
npm i

# Start application
npm run dev
```

**Notice: You should rename `src/config/index_bak.js` to `src/config/index.js` and update the config.**
